# Personal Portfolio Website — Raksha Verma

A responsive personal developer portfolio built with **HTML, CSS, and JavaScript**,
following professional coding practices and responsive design principles.

## 🚀 Overview
This site showcases my profile, skills, projects, and contact information, with a
friendly animated character in the hero section welcoming visitors.

## 🛠️ Technologies Used
- HTML5 (semantic elements)
- CSS3 (custom properties, Flexbox, Grid, media queries, keyframe animations)
- Vanilla JavaScript (DOM manipulation, events, form validation)
- Google Fonts (Space Grotesk, Inter, JetBrains Mono)

## ✨ Features
- Responsive layout for desktop, tablet, and mobile (with hamburger menu)
- Animated SVG character (a girl with curly hair) that waves hello on page load
  and re-waves periodically — used in place of a profile photo
- Typing animation cycling through role titles
- Dark / Light mode toggle (saved in localStorage)
- "Hire Me" button with an alert
- Contact form with client-side validation (name, email format, message length)
- Show / Hide "More About Me" toggle
- Smooth scrolling navigation + active-link highlighting on scroll
- Scroll-to-top button

## ▶️ How to Run
1. Download/clone the folder, keeping all files together.
2. Open `index.html` in any modern browser — no build step required.

## 🔧 Customize
- Update the name, About Me text, Education/Experience/Goals/Interests cards in `index.html`.
- Replace the 3 placeholder project cards with real projects, screenshots, and links.
- Update the email, LinkedIn, and GitHub links in the Contact section.
- Add real project screenshots into the `images/` folder and reference them in the project cards.
